使用说明：

1. 窗口参数

1.1. 协议名称
要注册的协议名称，比如cmd，注册后，是页面可以访问cmd://open的URL链接。

1.2. 关联程序
要用来打开协议的程序路径，可点击“浏览”按钮选择目标程序。

1.3. 使用参数
选择是否要将URL作为关联程序的启动参数。

2. 命令行参数

2.1. -p
要注册的协议名称，比如cmd，注册后，是页面可以访问cmd://open的URL链接。此选项指定后窗口中的对应项将不可更改。

2.2. -P
要用来打开协议的程序路径，如果路径中包含空格，须用半角双引号包裹路径。此选项指定后窗口中的对应项将不可更改。

2.3. -u
选择是否要将URL作为关联程序的启动参数，取值为yes或no。此选项指定后窗口中的对应项将不可更改。

3. 命令行启动
如果同时指定了所有的3个命令行参数，则直接注册URL协议。

4. Usage
UrlProtocolRegister.exe -p cmd -P "c:\windows\system32\cmd.exe" -u no
